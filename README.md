# Personal Portfolio Infrastructure (Static Frontend)

## Overview
This is my personal portfolio website built with semantic HTML5, external CSS, and JavaScript.
It includes:
- Hero section
- About me
- Technical skills (visual layout)
- Projects (3+ real projects)
- Context-based JavaScript calculator (Musanze Transport Fare Estimator)
- Downloadable CV + HTML preview
- Contact form (no backend) with validation
- Responsive design (mobile/tablet/desktop)
- Additional interactions: dark/light mode, dynamic footer year, mobile menu toggle

## Folder Structure
portfolio/
- index.html
- css/styles.css
- js/main.js
- assets/images/
- assets/cv/cv.pdf

## How to Run
Just open `index.html` in your browser.
